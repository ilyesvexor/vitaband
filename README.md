# VitaBand

VitaBand — Prototype repository. This repo contains a landing page (docs/), firmware scaffold for ESP32, a companion app scaffold (Flutter), and hardware notes (BOM, PCB notes).

الملف يحتوي على صفحة هبوط، واجهة اختبار للـ firmware والتطبيق، وملفات BOM وملاحظات PCB.

## Quick start
- To publish the landing page: enable GitHub Pages from Settings -> Pages, select branch 'main' and folder '/docs'.
- Firmware: see firmware/README.md
- App: see app/README.md
