Prototype snapshot

  • Core: ESP32-WROOM (BLE) — MAX30102 (HR & SpO₂) — BME680 (temp / humidity / VOC)
  • Battery: 3.7V LiPo, 300–500 mAh
  • Display: 1.3" 240×240 ST7789 (SPI)
  • Companion app: cross-platform (Flutter) — real-time BLE dashboard

This is an early prototype — not a medical device.

How it works

VitaBand measures photoplethysmography (PPG) using the MAX30102 sensor to estimate heart rate and SpO₂, computes HRV features on-device for privacy, and captures ambient environment metrics to provide contextualized insights.
